# ararat-gear

Мобильная страница со снаряжением для восхождения на Арарат. Автодеплой на GitHub Pages при пуше в `main`.

## Рабочий процесс для фото снаряжения (важно)

Владелец периодически кидает фотографии своей экипировки. По каждому фото нужно:

1. **Распознать**, что на фото (бренд/модель/тип снаряжения).
2. Определить, есть ли уже подходящий item в `data/gear.json`:
   - если есть — привязать фото к нему (`"photo": "<файл>.jpg"`);
   - если нет — добавить новый item в подходящую зону (`zones[].items[]`) со статусом (`have` / `buy` / `rent`) и осмысленным `name`/`notes`.
3. Положить обработанное фото в `images/` (повернуть по EXIF, сжать до ~1400px / ~q82, имя = `id` item, напр. `boots-trek.jpg`).
4. **Коммитить и пушить сразу в `main`** — НЕ создавать отдельную ветку и НЕ открывать PR. Это простые обновления контента, они должны накапливаться в `main`.

Цель — чтобы на сайте постепенно копилась вся экипировка владельца.

5. **После завершения** — прислать владельцу ссылку на страницу: <https://nawka.github.io/ararat-gear/>

## Структура

- `index.html`, `assets/style.css`, `assets/app.js` — страница
- `data/gear.json` — список снаряжения (зоны → items; статус `have` / `buy` / `rent`; поле `photo` = имя файла в `images/`)
- `images/` — фото снаряжения
- `.github/workflows/deploy.yml` — деплой на Pages
